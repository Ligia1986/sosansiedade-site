Política de Privacidade

Última atualização:

Respeitamos a tua privacidade. Esta Política explica que dados recolhemos, porquê, como os tratamos e os teus direitos ao abrigo do RGPD.

Quem somos

SOS Ansiedade é uma aplicação de bem-estar que fornece exercícios de respiração, SOS de pânico, meditações guiadas e um assistente com IA para apoio emocional.

Responsável pelo tratamento: SOS Ansiedade (suporte@sosansiedade.com).

Que dados recolhemos

  • Dados de utilização (anónimos/aglomerados): eventos básicos de uso (e.g., ecrãs usados, idioma, versão da app, sistema operativo).
  • Conteúdos facultados por ti na app (ex.: notas, textos introduzidos no chat de IA, preferências, favoritos). Podes apagar estes dados a qualquer momento na app, quando disponível.
  • Áudio (opcional): quando utilizas “Falar” (speech-to-text) ou “Ouvir” (text-to-speech). O áudio é processado para gerar texto/voz e não é usado para te identificar.
  • Notificações (opcional): token de notificação para enviar alertas/lembranças. Podes desativar nas definições do dispositivo.
  • Suporte: se nos contactares por email, recolhemos o conteúdo dessas mensagens para resolver o pedido.

Como usamos os dados (finalidades & bases legais)

  • Prestar o serviço (execução do contrato): mostrar conteúdos, guiar exercícios, gerar respostas com IA.
  • Melhoria e segurança (interesse legítimo): métricas de uso e registos de erro para estabilidade e prevenção de abuso.
  • Comunicações (consentimento/interesse legítimo): enviar notificações, responder a pedidos de suporte.

Tratamento por IA

Quando utilizas o chat/assistente, o texto que escreves é enviado para o nosso fornecedor de IA para gerar a resposta (ex.: OpenAI). Esses fornecedores atuam como subencarregados de tratamento e processam os dados apenas para prestar o serviço, com salvaguardas técnicas e contratuais. Não utilizamos estes conteúdos para treinar os nossos modelos sem o teu consentimento.

Partilha com terceiros (subprocessadores)

Poderemos usar serviços de:

  • Geração de texto/voz (IA, TTS/STT) para processar o teu input e devolver saídas.
  • Analytics e crash reporting para métricas anónimas e estabilidade.
  • Notificações para envio de alertas no dispositivo.

Estes fornecedores apenas tratam dados conforme instruções nossas e com medidas de segurança adequadas.

Transferências internacionais

Se os dados forem tratados fora do EEE, usamos salvaguardas legais adequadas (por ex., Cláusulas Contratuais-Tipo) para proteger os teus direitos.

Retenção

  • Dados locais na app: mantidos no teu dispositivo até seres tu a remover (ex.: notas, preferências).
  • Registos técnicos/erros: mantidos pelo tempo necessário para diagnóstico e segurança.
  • Pedidos de suporte: mantidos enquanto necessário para resolver o assunto e cumprir obrigações legais.

Os teus direitos (RGPD)

Tens direito a aceder, retificar, apagar, limitar, opor-te ao tratamento e portabilidade dos teus dados. Também podes retirar o consentimento quando aplicável.

Para exercer os direitos, contacta: suporte@sosansiedade.com. Responderemos dentro de prazos legais.

Segurança

Usamos medidas técnicas e organizativas adequadas (encriptação em trânsito, minimização de dados, controlos de acesso). Nenhum método é 100% infalível, mas trabalhamos para proteger os teus dados.

Crianças

A app destina-se a maiores de 16 anos. Se tens menos de 16, utiliza com o consentimento do teu responsável legal. Se achas que recolhemos dados de um menor sem consentimento, contacta-nos para remover.

Cookies e tecnologias semelhantes

No site poderemos usar cookies/armazenamento local para funcionalidades essenciais e métricas. Na app móvel, poderemos usar identificadores do dispositivo com finalidades equivalentes. Podes gerir preferências no navegador/dispositivo.

Alterações a esta Política

Podemos atualizar esta Política para refletir melhorias ou requisitos legais. Publicaremos a versão atualizada nesta página e ajustaremos a data de “Última atualização”.

Contacto

Dúvidas, pedidos ou reclamações: suporte@sosansiedade.com
